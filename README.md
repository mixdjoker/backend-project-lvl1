## Installation Guide

### GitHub installation

```sh
git clone git@github.com:mixdjoker/backend-project-lvl1.git
cd backend-project-lvl1
make install
```

[![asciicast](https://asciinema.org/a/318860.svg)](https://asciinema.org/a/318860)

### Uninstall

```sh
make uninstall
cd .. && rm -rf backend-project-lvl1
```
